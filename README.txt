For using the application, please follow the following steps:

1. Setting the database URL:

In order to use the application, you will have to set the database URL in the "DbUtilities.java" located at "src\waps\hw\vibhas"

The database (miniblog) is already created inside the project. You just need to edit the "dbPath" to the destination of the project.

2. You will need to run the project in a Tomcat server with a Derby database engine running.

3. Sequence Diagram are in the "Sequence Diagrams.docx" file.
